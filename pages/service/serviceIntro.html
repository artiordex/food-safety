식품안전 데이터 플랫폼 소개

'허브'는 중심점과 연결의 의미를 담고 있습니다.

식품안전 데이터 플랫폼은 공공데이터를 기반으로 대화를 통해
인사이트를 찾아주는 새로운 생성형 AI 서비스 입니다.

마치 교통의 허브가 여러 경로를 연결하고 여행자들이 모이는 중심지 역할을 하듯이,
식품안전나라의 다양한 공공데이터와 사용자를 일상 대화하듯 연결해
보다 손쉽게 정보를 찾아주는 새로운 경험을 느껴보세요.

본 서비스가 나날이 발전해 갈 수 있도록 다양한 의견과 피드백을 남겨주세요.

식품안전 데이터 플랫폼의 목적

통찰력 제공
시민, 연구자, 정책 입안자에게 식품안전의 현재와 미래를 제시하여 올바른 의사결정을 지원합니다.
스마트 행정 구현
데이터의 활발한 활용을 통해 보다 스마트하고 체계적인 식품안전 관리 시스템을 구축합니다.
국민 삶의 질 향상
데이터 기반 행정과 서비스 제공을 통해 먹거리 안전을 보장하고 국민의 삶의 질을 높이는 데 기여합니다.

식품안전 데이터 플랫폼의 핵심 가치

개방성
데이터 서비스가 필요한 모든 이에게 항상 개방되어 있는 플랫폼을 지향합니다.
접근성
데이터 활용에 익숙하지 않은 초심자도 대화형 인터페이스로 쉽게 접근이 가능합니다.
혁신
새로운 가치 창출을 위해 데이터 활용 방식에 대한 지속적이고 혁신적인 접근을 시도합니다.

식품안전 데이터 플랫폼의 핵심 서비스

식품안전 데이터 플랫폼에서는 데이터 서비스의 3요소인 “검색, 탐색, 활용” 서비스를 보다 세밀하고 고도화된 프리미엄 서비스로 업그레이드하여 제공합니다.

01. 데이터 검색원하는 데이터를 쉽고 정확하게 검색할 수 있습니다.

공공데이터 챗봇 서비스바로가기

데이터 검색에 어려움을 겪을 수 있는 초심자도 거부감 없이 일상 언어로 대화하듯 질문하고 답변을 받을 수 있습니다.
복잡한 쿼리 작성 없이 데이터 다운로드와 간단한 인사이트 도출이 가능한 새로운 경험을 제공합니다.

데이터 검색 서비스바로가기

데이터를 검색하기 위한 세부적인 검색 옵션을 제공하여 필요한 내용을 빠르게 찾을 수 있도록 도와줍니다.
데이터 검색 및 활용에 익숙한 전문가 사용자에게 더욱 유용한 기능입니다.

02. 데이터 탐색직관적인 방법으로 필요한 데이터를 빠르게 발견할 수 있습니다.

데이터 맵 서비스바로가기

수많은 데이터를 사용자의 눈높이에 맞게 분류하여 트리맵, 워드클라우드 등 시각화된 형태로 제공하는 탐색 서비스입니다.
관심 있는 분야를 직관적으로 파고들어 데이터를 발견할 수 있습니다.

데이터 설명서 (사전)바로가기

데이터셋이 가진 정보 유형, 정형성, 주요 컬럼 설명 등을 한눈에 파악할 수 있게 도표 형태로 제공합니다.
데이터 구조를 미리 이해하고 활용 파이프라인을 설계하는 데 필요한 안내서 역할을 합니다.

03. 데이터 활용다양한 시각화 콘텐츠를 통해 한 눈에 종합 정보를 확인할 수 있습니다.

주제분석 시각화바로가기

시민의 관심도가 높은 주요 식품안전 주제를 선정하여 동적인 인포그래픽 형태로 시각화한 콘텐츠입니다.
고정된 데이터가 아닌, 기간, 지역 등 변수를 변경하며 다각도로 지표를 분석할 수 있습니다.

3차원 시각화바로가기

학업이나 정책 연구에 유용한 주제의 데이터를 분석하여 3차원 애니메이션 등 고도화된 형태로 시각화합니다.
다차원 데이터를 융합 분석할 때 효과적인 탐색 도구를 제공합니다.
