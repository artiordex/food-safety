식품의약품안전처 데이터활용서비스

식품안전 공공데이터를 찾고, 연결하고, 활용하는 통합 안내

식품안전나라와 연계기관이 개방한 식품·건강 데이터를 Open API와 파일 형태로 제공합니다. 공공데이터포털의 검색·활용·제공신청 흐름을 참고해, 필요한 데이터를 빠르게 탐색하고 인증키 기반으로 서비스에 연결할 수 있도록 구성했습니다.

공공데이터 포털형 핵심 기능

데이터를 단순 나열하지 않고, 검색부터 신청·활용·공유까지 이어지는 이용 흐름을 한 화면에서 이해할 수 있습니다.

D

데이터 통합 탐색

식품영양, 위해관리, HACCP, 수입식품, 업체 인허가 등 식품안전 분야 데이터를 주제별로 탐색합니다.

M

데이터 구조 확인

서비스 ID, 컬럼, 요청 URL, 샘플 응답을 함께 확인해 개발 전에 데이터 구조를 먼저 파악합니다.

K

인증키 기반 이용

회원가입 후 인증키를 발급받고, 필요한 Open API 서비스에 이용 신청해 애플리케이션에 연결합니다.

U

활용 사례 확장

데이터활용 갤러리와 활용사례처럼 실제 서비스 아이디어를 축적하고 공유할 수 있는 구조를 지향합니다.

주요 데이터 분류

식품안전나라 데이터활용서비스의 분류 체계를 참고해 현장에서 자주 찾는 데이터 묶음을 먼저 보여줍니다.

식품정보

식품영양·기준규격

영양성분, 식품첨가물, 기준규격, 코드정보처럼 제품·성분 분석에 필요한 기초 데이터를 제공합니다.

안전관리

회수·판매중지·위해관리

부적합, 회수, 판매중지, 위해 식품 관련 정보를 활용해 소비자 알림과 모니터링 서비스를 만들 수 있습니다.

업체현황

인허가·HACCP·검사기관

업소 인허가, HACCP 지정, 검사기관 현황 등 기관·업체 기반의 관리 데이터를 확인합니다.

연계데이터

수입식품·축산물·위생용품

연계기관 데이터를 포함해 식품 공급망과 생활 안전 전반으로 활용 범위를 넓힙니다.

이용 절차

공공데이터포털의 제공신청 흐름처럼 찾는 데이터가 없을 때의 요청 경로까지 함께 안내합니다.

데이터 검색

분류별, 제공기관별, 유형별 메뉴에서 필요한 데이터와 서비스 ID를 찾습니다.

상세 구조 확인

요청 주소, 필수 파라미터, 응답 항목, 샘플 데이터를 확인합니다.

인증키 신청

회원 로그인 후 Open API 이용 신청을 진행하고 인증키를 발급받습니다.

서비스 연동

JSON 또는 XML 응답을 애플리케이션, 분석 도구, 업무 시스템에 연결합니다.

활용 공유

개발 결과와 활용 사례를 갤러리·사례 페이지에 등록해 재사용 가능성을 높입니다.

Open API 기본 형식

식품안전나라 Open API 호출 규칙을 개발자가 바로 이해할 수 있도록 핵심만 정리했습니다.

GET http://openapi.foodsafetykorea.go.kr/api/{keyId}/{serviceId}/{dataType}/{startIdx}/{endIdx}
keyId 발급받은 인증키
dataType json 또는 xml
startIdx/endIdx 조회 시작·종료 번호

운영 원칙과 이용 유의사항

공공데이터는 자유로운 활용을 지향하지만, 데이터별 이용 조건과 호출 제한을 확인해야 합니다.

활용 전 확인할 것

  • 데이터별 제공기관, 갱신 주기, 제공 형식, 이용허락 범위를 확인하세요.
  • 상업적 서비스에 적용할 경우 공공데이터 이용 조건과 출처 표기 방식을 확인하세요.
  • Open API 제한 운영, 서비스 점검, 항목 변경 공지는 반드시 사전에 확인하세요.

찾는 데이터가 없을 때

공공데이터포털처럼 원하는 데이터가 제공되지 않는 경우 제공신청이나 문의 경로를 통해 추가 개방을 요청할 수 있습니다. 본 서비스에서도 데이터 공개 범위와 연계기관 확장을 고려해 요청 사항을 관리합니다.
