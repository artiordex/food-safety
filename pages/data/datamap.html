식품안전나라 데이터맵

식품안전나라 데이터맵이란?

식품안전나라가 보유한 공개 가능한 데이터의 소재정보 및 연관관계를 쉽게 검색하여 찾아볼 수 있는 데이터 관계도입니다.

데이터 분포 트리맵 (Treemap)

트리맵이란?

각 분야별 데이터 건수를 면적 크기로 표현한 차트입니다. 면적이 클수록 해당 분야의 데이터가 많으며, 클릭하면 세부 데이터세트 목록을 볼 수 있습니다.

키워드 데이터맵 (D3 Force)

키워드 데이터맵이란?

특정 키워드와 연관된 데이터세트를 분야별로 분류하여 네트워크 그래프로 시각화합니다. 노드를 드래그하거나 스크롤로 확대/축소할 수 있습니다.

식품안전 데이터 융합 관계도

식품안전 데이터 융합 관계도란?

식품안전 데이터셋을 ERD 카드 형태의 노드로 표시하고, 공통 조인 키와 실제 매칭 관계를 포스 다이렉티드 네트워크로 시각화합니다.
