서비스 소개 서비스 이용방법

서비스 소개

식품의약품안전처 및 연계기관에서 공개하는 169종 공공데이터를 XML · JSON 형태의 Open API로 제공합니다.
누구나 인증키를 발급받아 데이터를 자유롭게 활용할 수 있습니다.

서비스 특징

📦

169종 공공데이터

식품안전, 영양정보, HACCP, 수입식품, 이력추적 등 식품안전 전 분야의 데이터를 통합 제공합니다.

🔌

XML · JSON 이중 포맷

다양한 서비스와 데이터를 좀 더 쉽게 이용할 수 있도록 XML과 JSON 두 가지 응답 포맷을 지원합니다.

🌐

RESTful API

직관적이고 표준화된 HTTP URL 방식을 사용하여 어떤 개발 환경에서도 쉽게 연동할 수 있습니다.

🏛️

11개 연계기관

식약처를 포함한 해양수산부, 농림축산검역본부, 농촌진흥청 등 11개 기관의 데이터를 한 곳에서 조회합니다.

API 기본 URL

GET http://openapi.foodsafetykorea.go.kr/api/{keyId}/{serviceId}/{dataType}/{startIdx}/{endIdx}
파라미터 타입 필수 설명
keyId String O 발급받은 인증키 (회원 가입 후 신청)
serviceId String O 서비스명 (예: I1250, C002)
dataType String O 응답 형식 (XML 또는 JSON)
startIdx Integer O 페이징 시작 인덱스 (1부터 시작)
endIdx Integer O 페이징 종료 인덱스

이용 절차

1
사이트 접속 및 회원가입

식품안전나라 데이터활용서비스에 접속하여 회원가입을 진행합니다. 서비스 이용을 위해서는 ID와 인증키가 필요합니다.

2
인증키 신청

원하는 API 서비스를 선택한 뒤 이용 신청 버튼을 클릭합니다. 활용용도를 반드시 작성하여 주십시오.

3
API 검색 및 이용방법 확인

서비스별 개발 명세서를 확인하고 포맷에 따라 목적에 맞게 활용하세요.

4
API를 이용하여 어플리케이션 제작

인증키를 포함한 API URL을 호출하여 XML/JSON 데이터를 받아 활용합니다.

5
어플리케이션 등록

제작한 서비스·앱을 데이터활용 갤러리에 등록하여 활용 사례를 공유할 수 있습니다.

이용 주의사항

  • 서비스를 사용하기 위해서는 ID와 인증키가 필요합니다.
  • 서비스 인증키 신청을 해주세요. 인증키는 타인에게 양도할 수 없습니다.
  • 서비스별로 개발 명세서에 설명되어 있는 포맷에 따라 목적에 맞게 활용하세요.
  • 인증키 신청하실 때 활용용도를 반드시 작성하여 주십시오.
  • 상업적 목적의 이용 시에는 공공데이터 이용약관을 반드시 확인하세요.
